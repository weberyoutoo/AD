1. Requirement:

	此.py檔請執行在anaconda environment下。

	* env的pachage的建議版本如下:

	Package                       Version  
	----------------------------- ---------
	torch                         1.10.1
	torchvision                   0.11.2
	tqdm                          4.64.1
	numpy                         1.19.5
	scikit-learn                  0.24.2
	scipy                         1.5.4
	matplotlib                    3.3.4
	pandas			      1.4.4

2. Run the hw1.py:

	(1)請先進入已安裝適當版本package的anaconda environment。
		eg. conda activate [your_env]

	(2)移動到hw1.py所在directory
		eg. cd [path to HW2_112062646]\HW2_112062646

	(3)執行
		python HW2.py

3. 備註:
	由於code中使用了plt.show()等api，所以可能會執行到一辦暫停，直到使用者將跳出的圖像關閉才會繼續執行。
